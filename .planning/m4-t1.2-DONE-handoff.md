  undefined `originalMimeType` ‚Üí fixed by hardcoding `'moq/mmt'` in the factory.

  proven end-to-end via `Player.load()`. NTP-short ~18.2h **wrap unwrapped**
  (threshold-guarded vs reorder). **Init-ordering hazard fixed** (RAP gate also
  requires the Init MPU stored, so the shared InitSegmentReference is never
  appended empty).
