  `isMmtp` branch; builds video Stream via shared `buildVideoStreamSkeleton_`
  (extracted, shared with LOC branch — DRY); `InitSegmentReference` empty
  (transmuxer supplies init); `mmtpFrameDuration_(track)` = `1/track.framerate`,
  **throws if absent** (CQ#1); subscribe callback: incomplete→noop, `fragment`→
  `shaka.log.alwaysError`+skip (NOT throw — async callback swallows throws),
  non-mfu→skip, RAP gate (drop until first `rapFlag===true`), startTime from
  `MmtpTimeline.toPresentationSeconds(rec.timestamp)` (NTP-short→sec, normalized),
  per-frame `SegmentReference`+`setSegmentData`, `mergeAndEvict` (bounded live).
  `createSegmentIndex` uses `Promise.withResolvers` (resolves on first segment).
  Video-only `goog.asserts` guard at entry.
