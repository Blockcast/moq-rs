# M.4 T1.2 — MMTP Video Playback Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Render real FFmpeg `moq_mmt` MMTP multicast video end-to-end in Shaka MSE, replacing the observe-only dump with a real transmux → SegmentReference → playback path.

**Architecture:** A dedicated `shaka.transmuxer.MmtpTransmuxer` (a `shaka.extern.Transmuxer`, registered for `moq/mmt`, converting to `video/mp4`) mirrors `LocTransmuxer.getAvcStreamInfo_`: it caches the H.264 `videoConfig` from the first IDR's in-band SPS/PPS and uses `Mp4Generator` to emit `{init, media}` per frame. A new `processMmtpTrack_` in `msf_parser.js` mirrors the LOC branch — reassembles MFUs, gates on the first RAP frame, derives `startTime` from the decoded per-MFU MMTP timestamp, and emits one evicting `SegmentReference` per frame. A separate cross-repo fix corrects the `source_FEC_payload_ID` prefix→trailer parity bug.

**Tech Stack:** Shaka Player (Closure JS), `Mp4Generator`/`H264` transmux utils, Karma + `build/test.py`; Rust `mmt_core` + `moq-pub-mmtp` example generator for the parity vector.

**Spec:** `.planning/m4-t1.2-video-playback-design.md` (rev2 + D9 revert to in-band SPS/PPS). Decisions: Arch#1 (thin-delegating), CQ#1 (timestamp timing), CQ#2/T4 (FEC trailer), D-perf1 (eviction), D-seg (per-frame), D-mime, D-seq (first-RAP gate), D-bframe (no ctts), D9 (in-band config — mmpu-strip dropped).

**Decisions superseded by D9:** the rev2 spec's D-init / mmpu-strip (T2) and "transmuxer media-only" are REVERTED. Init is synthesized from in-band SPS/PPS (LocTransmuxer pattern). No catalog init, no fallback.

```js
// lib/transmuxer/mmtp_transmuxer.js
goog.provide('shaka.transmuxer.MmtpTransmuxer');
goog.require('shaka.transmuxer.H264');
goog.require('shaka.transmuxer.TransmuxerEngine');
goog.require('shaka.transmuxer.TransmuxerUtils');
goog.require('shaka.util.BufferUtils');
goog.require('shaka.util.ManifestParserUtils');
goog.require('shaka.util.MimeUtils');
goog.require('shaka.util.Mp4Generator');
goog.require('shaka.util.StreamUtils');
goog.requireType('shaka.media.SegmentReference');

/**
 * Transmuxes MMTP MFU video (AVCC length-prefixed H.264 NAL) into fMP4 for MSE.
 * Mirrors LocTransmuxer's AVC path; codec config is cached from the first IDR's
 * in-band SPS/PPS (decision D9). MMTP-only, video-only (Sub-project A).
 * @implements {shaka.extern.Transmuxer}
 * @export
 */
shaka.transmuxer.MmtpTransmuxer = class {
  /** @param {string} mimeType */
  constructor(mimeType) {
    /** @private {string} */
    this.originalMimeType_ = mimeType;
    /** @private {number} */
    this.frameIndex_ = 0;
    /** @private {!Map<string, !Uint8Array>} */
    this.initSegments = new Map();
    /** @private {?Uint8Array} */
    this.lastInitSegment_ = null;
    /** Cached H264.parseInfo() from the first IDR carrying SPS+PPS.
     * @private {?{videoConfig: !Uint8Array, hSpacing: number, vSpacing: number,
     *            height: number, width: number}} */
    this.avcInfo_ = null;
  }

  /** @override @export */
  destroy() {
    this.initSegments.clear();
    this.avcInfo_ = null;
  }

  /** @param {string} mimeType @return {boolean} @private */
  isMmtpContainer_(mimeType) {
    return mimeType.toLowerCase().split(';')[0] == 'moq/mmt';
  }

  /** @override @export */
  isSupported(mimeType, contentType) {
    const ContentType = shaka.util.ManifestParserUtils.ContentType;
    if (!this.isMmtpContainer_(mimeType)) {
      return false;
    }
    // Sub-project A is video-only.
    if (contentType && contentType != ContentType.VIDEO) {
      return false;
    }
    return shaka.media.Capabilities.isTypeSupported(
        this.convertCodecs(ContentType.VIDEO, mimeType));
  }

  /** @override @export */
  convertCodecs(contentType, mimeType) {
    if (this.isMmtpContainer_(mimeType)) {
      const codecs = shaka.util.MimeUtils.getCodecs(mimeType).split(',')
          .map(shaka.util.StreamUtils.getCorrectVideoCodecs).join(',');
      return `video/mp4; codecs="${codecs}"`;
    }
    return mimeType;
  }

  /** @override @export */
  getOriginalMimeType() {
    return this.originalMimeType_;
  }

  /** @override @export */
  transmux(data, stream, reference, duration, contentType) {
    const uint8ArrayData = shaka.util.BufferUtils.toUint8(data);
    let streamInfo;
    try {
      streamInfo = this.getAvcStreamInfo_(
          uint8ArrayData, stream, reference, duration);
    } catch (e) {
      return Promise.reject(e);
    }
    if (!streamInfo) {
      // No cached config yet (pre-IDR) — emit nothing; caller gates on RAP.
      return Promise.resolve({data: new Uint8Array([]), init: null});
    }
    const mp4Generator = new shaka.util.Mp4Generator([streamInfo]);
    const initSegmentKey = stream.id + '_' + reference.discontinuitySequence;
    const initSegment = this.initSegments.has(initSegmentKey) ?
        this.initSegments.get(initSegmentKey) :
        this.initSegments.set(initSegmentKey, mp4Generator.initSegment())
            .get(initSegmentKey);
    const appendInitSegment = this.lastInitSegment_ !== initSegment;
    const segmentData = mp4Generator.segmentData();
    this.lastInitSegment_ = initSegment;
    this.frameIndex_++;
    return Promise.resolve(
        {data: segmentData, init: appendInitSegment ? initSegment : null});
  }

  /** @param {!Uint8Array} data @param {shaka.extern.Stream} stream
   * @param {?shaka.media.SegmentReference} reference @param {number} duration
   * @return {?shaka.util.Mp4Generator.StreamInfo} @private */
  getAvcStreamInfo_(data, stream, reference, duration) {
    const H264 = shaka.transmuxer.H264;
    const timescale = shaka.transmuxer.MmtpTransmuxer.VIDEO_TIMESCALE_;
    const nalus = this.extractAvccNalus_(data);
    const parsedInfo = H264.parseInfo(nalus);
    if (parsedInfo) {
      this.avcInfo_ = parsedInfo;
      stream.height = parsedInfo.height;
      stream.width = parsedInfo.width;
    }
    if (!this.avcInfo_) {
      return null; // pre-IDR: no config cached yet
    }
    const isKeyframe = H264.isKeyframe(nalus);
    const baseMediaDecodeTime = Math.floor(reference.startTime * timescale);
    const sampleDuration =
        Math.floor((reference.endTime - reference.startTime) * timescale);
    /** @type {!Array<shaka.util.Mp4Generator.Mp4Sample>} */
    const samples = [{
      data, size: data.byteLength, duration: sampleDuration, cts: 0,
      flags: isKeyframe ? shaka.transmuxer.TransmuxerUtils.VIDEO_KEYFRAME_FLAGS :
          shaka.transmuxer.TransmuxerUtils.VIDEO_NON_KEYFRAME_FLAGS,
    }];
    return {
      id: stream.id, type: shaka.util.ManifestParserUtils.ContentType.VIDEO,
      codecs: 'avc1', timescale, duration,
      mediaConfig: this.avcInfo_.videoConfig,
      hSpacing: this.avcInfo_.hSpacing, vSpacing: this.avcInfo_.vSpacing,
      data: {sequenceNumber: this.frameIndex_, baseMediaDecodeTime, samples},
      stream,
    };
  }

  /** Length-prefixed (AVCC) NAL extraction — MMTP MFU payload is always AVCC.
   * @param {!Uint8Array} data @return {!Array<shaka.extern.VideoNalu>} @private */
  extractAvccNalus_(data) {
    const nalus = [];
    let offset = 0;
    while (offset + 4 <= data.byteLength) {
      const len = ((data[offset] << 24) | (data[offset + 1] << 16) |
          (data[offset + 2] << 8) | data[offset + 3]) >>> 0;
      offset += 4;
      const end = offset + len;
      if (end > data.byteLength || end <= offset) {
        break;
      }
      const fullData = data.subarray(offset, end);
      const type = fullData[0] & 0x1f;
      const naluData = fullData.length > 1 ?
          fullData.subarray(1) : new Uint8Array([]);
      nalus.push({type, data: naluData, fullData});
      offset = end;
    }
    return nalus;
  }
};

/** Standard MP4 video timescale (ticks/sec). @private @const {number} */
shaka.transmuxer.MmtpTransmuxer.VIDEO_TIMESCALE_ = 90000;

shaka.transmuxer.TransmuxerEngine.registerTransmuxer(
    'moq/mmt',
    (mimeType) => new shaka.transmuxer.MmtpTransmuxer(mimeType),
    shaka.transmuxer.TransmuxerEngine.PluginPriority.PREFERRED_SECONDARY);
```

/** Converts per-MFU NTP-short MMTP timestamps to a normalized presentation
 * timeline (first accepted timestamp == 0). NTP-short = 16.16 fixed point. */
shaka.msf.MsfParser.MmtpTimeline = class {
  constructor() {
    /** @private {?number} */
    this.baseSeconds_ = null;
  }
  /** @param {number} ntpShort @return {number} seconds from live edge */
  toPresentationSeconds(ntpShort) {
    const seconds = (ntpShort >>> 0) / 65536; // 2^16
    if (this.baseSeconds_ === null) {
      this.baseSeconds_ = seconds;
    }
    return seconds - this.baseSeconds_;
  }
};

  const ContentType = shaka.util.ManifestParserUtils.ContentType;
  const codec = (track.selectionParams && track.selectionParams.codec) ||
      track.codec;
  const mimeType = `moq/mmt; codecs="${codec}"`;

  const initSegmentReference = new shaka.media.InitSegmentReference(
      () => [], 0, null, null, timescale, new Uint8Array([]));
  const processor = new shaka.msf.MmtpTrackProcessor(track.maxBufferedMfus);
  const timeline = new shaka.msf.MsfParser.MmtpTimeline();
  let sawRap = false;

  stream.createSegmentIndex = () => {
    this.subscribeToTrack(track, trackKey, (obj) => {
      if (!stream.segmentIndex || !obj.data.byteLength) {
        return;

      const reference = new shaka.media.SegmentReference(

      this.presentationTimeline_.notifyMaxSegmentDuration(frameDuration);
      this.receivedFirstSegment_.add(ContentType.VIDEO);
    });
    return Promise.resolve();
  };
